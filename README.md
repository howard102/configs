# Configs

This repository stores my personal development environment settings.

## Environment

- Windows Terminal
- PowerShell 7
- Ubuntu WSL2
- Git
- Node.js
- pnpm
- Codex / GPT assisted development
- Claude code 
- Gemini assisted development

## Design Principles

1. PowerShell is used for Windows management.
2. Ubuntu WSL2 is used for actual web development.
3. Project files should live under `~/projects`.
4. Config files should live under `~/configs`.
5. Do not use dangerous recursive delete commands.
6. Do not commit API keys or `.env` files.
